# GRUS — Citizen Unified Portal

This repository contains the citizen-facing web application for the **Governance Real-time Update System (GRUS)**. It provides an inclusive, transparent platform where citizens can track public service statuses and access verified community information.

## 🚀 Key Features
- **Live Office Tracker:** Shows real-time availability of local public offices (Active/Closed) directly updated by field officials.
- **Verified Community Q&A:** A dynamic space for citizens to submit questions and read authenticated answers from officials.
- **Unified Logic:** Syncs with the exact same centralized database used by our IVR voice bot to keep data completely consistent.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend Link:** FastAPI (Python)
- **Database:** PostgreSQL
